## Intrucciones compilación:

_Prerequisitos:_
* Node >= 16.13.1
* npm >= 8.1.2

Inicio Rápido: 

Dirigase a la ruta donde se encuentra el proyecto y en su terminal ejecute:
```
npm install
```

Y posteriormente:
```
npm start
```
